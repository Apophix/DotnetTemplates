# WebProject Solution Template 

## Backend 
- .NET 10 Web API
- Modular Solution Achitecture 
- Aspire Orchestration
- Centralized Package Management
- FastEndpoints
- OpenAPI + tweaks for apx.rest client generation
- EF Core with SQL Server and Postgres providers (just pick what you need)


## Frontend
- Tanstack Start in SPA Mode
- apx.rest client
- Typescript utilities like `use-streaming-request.ts` 


### Future State
- Built-in Fitness Testing 
- Parameterization to control "SPA mode"


### Installation

1. Clone the repository
2. cd into the project directory
3. Run `dotnet new install .` to install the template locally
